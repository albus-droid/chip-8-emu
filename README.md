# Chip-8 Emulator

A simple Chip-8 emulator built from scratch, heavily inspired by [Austin Morlan's Chip-8 Emulator Guide](https://austinmorlan.com/posts/chip8_emulator/). This emulator aims to run Chip-8 programs (ROMs) on modern systems, mimicking the behavior of the original 8-bit Chip-8 architecture.
